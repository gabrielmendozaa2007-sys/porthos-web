# Porthos · Web Chat

Página de demostración con el widget de chat de soporte de **Porthos Steakhouse & Pub**.

Sitio estático: un launcher flotante con el emblema de Porthos abre un chat conectado
a un agente de atención al cliente (responde preguntas frecuentes, captura el nombre
del visitante y redirige pedidos/reservas a WhatsApp).

## Archivos
- `index.html` — página + widget de chat (HTML/CSS/JS en un solo archivo)
- `porthos-mark.png` — emblema (estrella) usado en el launcher y el header
- `porthos-logo.png` — logo completo (referencia)

## Deploy
Sitio 100% estático. En Vercel: importar el repo y desplegar (sin build, output = raíz).
